# Odev3 – TypeScript Ödevi

Bu repo, 20 soruluk TypeScript ödevinin çözümlerini içerir.  
Her dosya bağımsızdır ve terminal üzerinden çalıştırılabilir.  
Çıktılar `"Soru X Başarılı!"` formatında olup ekran görüntüsü (SS) ile doğrulanabilir.

| Dosya Adı   | Açıklama |
|------------|----------|
| Soru1.ts   | Sabit değer döndüren basit fonksiyon |
| Soru2.ts   | Jenerik dizi filtreleme |
| Soru3.ts   | Jenerik dizi birleştirme |
| Soru4.ts   | Nesneden belirli property’leri çıkarma |
| Soru5.ts   | Jenerik key çıkarımı |
| Soru6.ts   | Jenerik key’e göre eleman seçme |
| Soru7.ts   | Maksimum değer bulma |
| Soru8.ts   | Dizi içindeki tekrarları kaldırma |
| Soru9.ts   | Alt küme kontrolü |
| Soru10.ts  | Property’lere göre gruplama |
| Soru11.ts  | Promise içeriğini çıkaran tip (`UnwrapPromise`) |
| Soru12.ts  | Fonksiyon dönüş tipi çıkarımı (`GetReturnType`) |
| Soru13.ts  | Fonksiyon parametre tipi çıkarımı (`GetParameters`) |
| Soru14.ts  | Jenerik `Partial<T>` benzeri yapı |
| Soru15.ts  | Jenerik `Required<T>` benzeri yapı |
| Soru16.ts  | Jenerik `Readonly<T>` benzeri yapı |
| Soru17.ts  | Jenerik `Pick<T, K>` benzeri yapı |
| Soru18.ts  | Jenerik `Omit<T, K>` benzeri yapı |
| Soru19.ts  | Union içinden belirli tipleri filtreleme |
| Soru20.ts  | Union içinden belirli tipleri dışlama |
